# 現代物理學的經典註腳

這個倉庫保存了本人30余年學習的過程中斷斷續續積累下來的現代科學和古代典籍的一些資料。竊以為，現代科學，特別是物理學，在先人的古典中已經有所蘊含，甚至於某些方面，現代物理學似乎剛剛觸摸到古人智慧的邊緣。

中學階段我參加了物理的奧賽，在興趣學習小組上，小夥伴們討論的最多的話題就是相對論，比如“天上一天，地上一年”、“不存在絕對的時間和空間”等不可思議的物理學現象。本科階段必修大學物理，進一步了解了量子力學的基礎知識，一直到現在，自己也在不斷地關註物理學界的一些最新發表的成果。

在一直對物理學保持關註的狀態下，也零星接觸了傳統文化中的一些經典。特別是初次讀到楞嚴、楞伽時，內心是震撼的，不由驚嘆先賢的認知水平。幾千年前雖然不存在現代科學這套語言系統，但經典裏面的很多的名相和概念都是暗契的。經典中也存在著一些關於世界觀上的論述，與現代科學觀測到的宇宙有較大的出入，但是我們要懷有開闊、敬畏之心，畢竟現代科學還非常年輕，不然會落入唯科學主義的窠臼。我們的感官和認知在整個信息域中實際非常狹窄，無法感知牛頓經典力學和宏觀尺度之外的自然現象，猶如井底之蛙。

這個倉庫不放未經科學論證的內容，只羅列一些現代科學的公認的理論和實驗可重復的現象，也包括一些可能存在爭議的科學家和哲學家的訪談或見解。
